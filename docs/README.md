# JeevanDhara

Hackathon MVP repository. This is the repo skeleton. See `/docs/spec.md` for the full specification.

Quickstart (minimal):
- Backend: `cd backend && npm install && npm start` then open `http://localhost:3000/api/health`.
- ML: `cd ml && pip install -r requirements.txt && python predict_service.py` then open `http://localhost:5000/`.
- Mobile: `cd mobile && flutter pub get && flutter run`.
- Docker compose: `cd infra && docker-compose up --build`.
