# JeevanDhara Monorepo

This repository contains the mobile app (Flutter), backend (Node.js + Express), ML microservice (Python + Flask), and infrastructure (Docker) for the JeevanDhara hackathon MVP.

- mobile: Flutter app
- backend: Node.js + Express API
- ml: Python Flask microservice
- infra: Dockerfiles and docker-compose
- docs: Documentation, spec, Postman collection, and env examples

Run instructions will be added as the project evolves.
